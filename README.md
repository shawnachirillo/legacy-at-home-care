# Legacy At Home Care Website

A polished starter website for **Legacy At Home Care**, built with:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Google Fonts: Lora + Work Sans

## Brand Direction

Warm, familiar, home-centered, trustworthy, and calming.

Core feeling: **Care that feels like home.**

## Colors

- Cream: `#F7F3EB`
- Linen: `#EFE7D8`
- Morning Sun: `#E6D19A`
- Sage: `#A6B09B`
- Wood: `#8C6A4A`
- Evergreen: `#4B5A4B`

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## Pages Included

- Home
- About
- Services
- Caregivers
- FAQ
- Contact

## Important Notes

The contact form is currently visual only. To make it functional, connect it to:
- Formspree
- Resend
- EmailJS
- HubSpot
- A custom Next.js API route
